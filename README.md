# Third Person Parkour and Freeflow Combat
Third-person character controller featuring combat and parkour. Using a hierarchical state machine and Unity Mecanim.

### Movement
- Movement and camera orbit
- Dashing
- Climbing up walls
- Wallrun
- Sliding down ramps

### Combat
- Freeflow system
- TargetGroup camera
- Visual feedback
